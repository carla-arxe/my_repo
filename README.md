# My Repo - Utilities and Automations for the CDM

This repository is a collection of practical scripts, guides, and utilities designed to optimize daily workflows and automate processes.

---

## 📁 Repository Structure

The repository is organized into three main workspaces:

### 1. [connection_setup](./connection_setup)
This folder contains instructions and a script to safely automate database connections at the hospital.

### 2. [phenotypeR](./phenotypeR)
A guide with clear explanations and examples on using the official OHDSI **PhenotypeR** package.

### 3. [Cohorts](./Cohorts)
A step-by-step tutorial detailing how to import a configuration file in **JSON** format (exported from ATLAS).
